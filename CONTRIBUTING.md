# Contributing

Thank you for considering contributing to PHP REST API Framework!

## Development Setup

1. Clone the repository
2. Install dependencies: `composer install`
3. Run tests: `composer test`
4. Run static analysis: `composer stan`

## Code Style

- Follow PSR-12 coding standards
- Use PHP 8.2+ features
- Add type hints everywhere
- Write tests for new features

## Testing

- Write tests for all new features
- Ensure all tests pass: `composer test`
- Aim for high test coverage

## Pull Requests

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Add tests
5. Ensure all tests pass
6. Submit a pull request

## Reporting Issues

Please use GitHub issues to report bugs or suggest features.
